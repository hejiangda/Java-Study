## 直接用JDK自带工具编译和运行程序
- 使用`javac`将`*.java`文件编译成字节码文件`*.class`
- 使用`java *`命令可以运行`*.class`

## 目录说明

**JavaBasic** 文件夹中放的是学《Java程序设计基础（第6版）》时写的练习代码，这本书讲的是Java的基本语法，它后面几章有些过时了，前面的基础部分讲得很清楚。

**CoreJavaVolume1** 文件夹中放的是学《Core Java Volume I》这本书时写的练习代码。
